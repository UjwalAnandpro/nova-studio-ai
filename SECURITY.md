# Security Policy

## Supported Versions

Only the latest active release versions of Nova Studio AI receive security updates.

| Version | Supported |
| ------- | --------- |
| v1.0.x  | Yes       |
| < v1.0  | No        |

## Reporting a Vulnerability

If you discover a security vulnerability, please do **NOT** open a public GitHub issue. Instead, report it responsibly by contacting the repository maintainers directly.

Once reported, we will evaluate the issue and work to prepare a patch release before disclosing the details publicly.
