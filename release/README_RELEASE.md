# Nova Studio AI v1.0.0 Release Package

Welcome to the production release of **Nova Studio AI (v1.0.0)**. 

## Package Contents

- **`NovaStudioAI_Setup.exe` (Standalone Installer)**: *Note: Compiled executable wizard binary setup installer is attached directly as a release asset in the GitHub Releases tag for repository size management.*
- **`LICENSE`**: MIT open-source license copy.
- **`CHANGELOG.md`**: Historical version additions list.
- **`CHECKSUMS.md`**: SHA-256 validation fingerprints.
- **`RELEASE_NOTES.md`**: Release feature summaries.

## Getting Started

1. Download **`NovaStudioAI_Setup.exe`** from the [GitHub Releases Page](https://github.com/UjwalAnandpro/nova-studio-ai/releases).
2. Double-click the installer and follow the wizard instructions.
3. Use the desktop shortcut to run the creative editor.
