# Nova Studio AI Development Roadmap

This roadmap documents the architectural expansion path of the local creative suite.

## Phase 1: Core Automation (v1.0.0 - Current)
- [x] Event-Driven core engine integrations.
- [x] Non-linear editor splits, clones, and deletions.
- [x] Speech phoneme subtitle timing compilation.
- [x] Dynamic python loader for category providers.

## Phase 2: AI Avatar & Lip Sync (Upcoming)
- [ ] **Viseme Waveform matched head-generation**: Extract sound waves parameters and interpolate custom talking avatar frames.
- [ ] **AI Voice Cloning**: Local audio sample overrides.

## Phase 3: Translation Dubs & Nodes (Future)
- [ ] **Subtitles dub translators**: Translating scripts to target languages and re-synthesizing voiceovers.
- [ ] **Distributed Rendering**: Farm out rendering chunks to local LAN servers.
