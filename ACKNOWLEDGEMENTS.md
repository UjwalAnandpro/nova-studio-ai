# Acknowledgements

Nova Studio AI stands on the shoulders of the open-source community. We want to thank the authors and maintainers of the following crucial foundations:

- **ComfyUI**: Node-based graph orchestrator execution backend.
- **Kokoro-82M**: Local phoneme-level speech synthesizer.
- **Streamlit**: Modern and interactive web layout application frame.
- **FFmpeg**: Engine powering video transcoders, subtitle integrations, and audio compilers.
- **SQLite**: Database management engine.
- **PyInstaller** and **Inno Setup**: Distributable packaging utilities.
